## Run Locally

Clone o repositorio

```bash
  git clone https://github.com/ViniVMA/Rockteer-Product-LandingPage
```

Va para o repositorio do projeto

```bash
  cd Rockteer-Product-LandingPage
```

Instale as dependencias

```bash
  yarn
```

Inicie o servidor local

```bash
  yarn dev
```
